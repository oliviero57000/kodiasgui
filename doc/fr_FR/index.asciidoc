= Kodiasgui v0.90

== Description
Ce plugin vous permet de vous interfacer avec des Kodi afin de les utiliser comme interface graphique.

Il est compatible avec le nouveau fonctionnement maître/esclave (plugin Jeedom Link) de Jeedom.

Pour fonctionner il faut impérativement installer un plugin et un skin specifique sur le Kodi.

Seul la version 16 de Kodi est supportée. Le plug-in Kodi peu fonctionner sur l'ensemble des plateformes supportée par Kodi.
